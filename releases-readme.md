# MeshtasticAI

A graphical interface for Meshtastic mesh networking devices with AI integration.

## Downloads

**[View All Releases](https://github.com/rdmcnabb/MeshtasticAI-Releases/releases)**

| Platform | Download |
|----------|----------|
| **Windows** | `MeshtasticAI-x.x.x-windows.exe` |
| **Linux** | `MeshtasticAI-x.x.x-linux` |

## Installation

### Windows
1. Download the `.exe` file from Releases
2. Double-click to run (no installation needed)

### Linux
1. Download the Linux file from Releases
2. Make it executable: `chmod +x MeshtasticAI-*-linux`
3. Run: `./MeshtasticAI-*-linux`

## Requirements

- A Meshtastic device connected via USB, TCP, or Bluetooth
- (Optional) Ollama for AI features
