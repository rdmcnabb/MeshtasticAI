  ## About                                                                      
                                                                                
  A graphical interface for Meshtastic mesh networking devices with AI          
  integration.                                                                  
                                                                                
  ## Requirements                                                               
                                                                                
  **Ollama is required for this application to work.** You must install Ollama  
  before running MeshtasticAI.                                                  
                                                                                
  1. Download and install Ollama from [https://ollama.com](https://ollama.com)  
  2. Start Ollama and ensure it is running                                      
  3. A Meshtastic device connected via USB, TCP, or Bluetooth

  ## Installation                                                               
                                                                                
  ### Windows                                                                   
  1. Download the `.exe` file from Releases                                     
  2. Double-click to run (no installation needed)                               
                                                                                
  ### Linux                                                                     
  1. Download the Linux file from Releases                                      
  2. Make it executable: `chmod +x MeshtasticAI-*-linux`                        
  3. Run: `./MeshtasticAI-*-linux`
